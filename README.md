# Team-final

1. 주제 : 꽃 배달 

2. 기능적 요구사항
- 고객이 꽃배달 메뉴 선택하여 주문한다
- 주문이 벌어지면 결제 시스템의 결제 기능이 호출된다
- 주문이 되면 주문 내역이 꽃가게 주인에게 전달된다
- 꽃가게 주인이 확인하여 꽃을 만들어서 배달 출발한다
- 고객이 주문을 취소할 수 있다
- 주문이 취소되면 배달이 취소된다
- 고객이 주문상태를 중간중간 조회한다
- 주문상태가 바뀔 때 마다 카톡으로 알림을 보낸다
- 꽃가게 주인이 꽃배달 메뉴를 관리한다
